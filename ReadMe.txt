Прога комбинирует данные из csv и выбранного рекламного кабинета в таблицу

Для ее работы нужно получить ключ доступа к ВК:
Перейди по URL:
https://oauth.vk.com/authorize?client_id=6604233&redirect_uri=https://oauth.vk.com/blank.html&display=page&scope=ads,offline&response_type=token&v=5.80
Разреши доступ, вк перекинет на страницу с текстом:
"Пожалуйста, не копируйте данные из адресной строки для сторонних сайтов.
Таким образом Вы можете потерять доступ к Вашему аккаунту."
Скопируй из URL текст после "access_token=" и до символа "&" (амперсанд).
Нужная часть выглядит примерно так:
10b9f8be5c03abe5de795b99343516dafcefabcc74b3ab4d3da9a3d52235b2b3ba2491cccf533358aae8d
Теперь просто вставь эту строку в файл settings.js после
const access_token = '
чтобы получилось что-то вроде
const access_token = '10b9f8be5c03abe5de795b99343516dafcefabcc74b3ab4d3da9a3d52235b2b3ba2491cccf533358aae8d';
Готово, сохраняй файл settings.js и открывай index.html в твоем браузере.

Выбираешь сверху рекламный кабинет и выгруженный из Анкет csv файл и она их комбинирует.
Принцип работы:
Оставляет только цифры в utm 1:
была utm1: "s1" стала: "1"
Ищет цифр из utm 1 во всех кампаниях аккаунта.
Связывает кампании с записями о лидах.
Затем в названиях всех активных объявлениях этой кампании ищет
соединение utm 1 и utm 2, и связывает объявления с записями.
Если этот текст будет в названии нескольких объявлений, то скрипт суммирует их траты.

Программа работает с агентскими кабинетами. Выбираешь какой-либо агентсткий кабинет.
Она загружает клиентов и:
если клиент только 1, то сразу же выбирает его и работает с ним,
иначе нужно выбрать клиента и она будет работать с ним.

Можно выбрать период сбора статистики с помощью календаря снизу.
Если он не выбран, то программа будет работать со статистикой за все время.
